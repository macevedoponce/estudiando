## Installation

Clonar Repositorio

```bash
git clone https://github.com/macevedoponce/estudiando.git
cd estudiando
```
Pasos importantes para el uso
```bash
npm install
composer install
```
Configurar archivo .env con credenciales y nombres de la BD
```bash
php artisan migrate
php artisan serve 
```
